## üéØ Overview

**CricDR Coach** is a revolutionary web application that bridges the gap between aspiring cricketers and professional coaching. Built with modern web technologies, it provides a comprehensive platform for cricket training management, progress tracking, and AI-powered performance analysis.

I created CricDR Coach as a virtual coaching platform specifically tailored for beginner cricketers. ## ü§ñ AI Service & Logic

### **Performance Analysis Engine**

#### **Metrics Calculation**
```typescript
// Consistency Score Algorithm
const calculateConsistency = (entries: ProgressEntry[]): number => {
  const practiceCount = entries.filter(e => e.netHours > 0).length;
  return (practiceCount / entries.length) * 100;
};

// Improvement Tracking
const calculateImprovement = (entries: ProgressEntry[]): number => {
  const recent = entries.slice(-3);
  const older = entries.slice(-6, -3);
  const recentAvg = recent.reduce((sum, e) => sum + e.selfRating, 0) / recent.length;
  const olderAvg = older.reduce((sum, e) => sum + e.selfRating, 0) / older.length;
  return ((recentAvg - olderAvg) / olderAvg) * 100;
};
```
